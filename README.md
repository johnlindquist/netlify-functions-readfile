# Example for accessing local files in Netlify Functions

## Try it out!

### Via `netlify-cli`

```
npm -g i netlify-cli
netlify sites:create --name my-test-site
netlify link
netlify deploy --prod
```

### Via netlify build

1. Fork this repository
1. Go to https://app.netlify.com/start
1. Create a site from your fork
